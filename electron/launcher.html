配置 Python 微服务

PaperSpark 需要依赖 Python 环境来运行本地 OCR、文档分析和 Python 在线运行服务。请选择一个包含所需依赖的运行环境。

已检测到的环境
Modal 云部署（最推荐）
适合希望获得更稳定 OCR 性能、较低本地环境负担，同时又不想长期维护 GPU 服务的用户。
首次使用 Surya OCR 时需要下载模型并建立缓存，启动时间会明显更长，请耐心等待。部署后把服务地址填到下面，PaperSpark 会直接连接云端服务。
1
在 Modal 部署 Surya OCR 按照 README 的部署步骤，拿到一个类似 `https://your-name--paperspark-surya-web-app.modal.run` 的地址。
2
把地址填进 PaperSpark 把云端地址填写到下面输入框，PaperSpark 会把它写入运行配置。
3
直接启动工作台 启动后，论文导入解析、知识库精读和相关 OCR 能力会直接走云端服务。
支持 `https://` 或 `http://` 地址，建议直接粘贴 Modal 部署成功后打印的 URL。 SURYA_OCR_SERVICE_URL SURYA_SERVICE_URL NEXT_PUBLIC_SURYA_SERVICE_URL NEXT_PUBLIC_SURYA_OCR_SERVICE_URL
